CÓMO ACTUALIZAR EL JUEGO
  1. Edita equipos y jugadores en las pestañas de arriba
  2. Pulsa Exportar data.json — se descarga el archivo
  3. Copia ese archivo a public/data/data.json en tu proyecto
  4. Ejecuta git add . && git commit -m "datos" && git push
  5. Vercel redespliega en ~30 segundos con los nuevos datos
