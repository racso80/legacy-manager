# Escudos de equipos

Coloca aquí los escudos con fondo transparente y el ID del equipo como nombre:

- `athletic.png`
- `barcelona.png`
- `realsociedad.png`

Se recomienda PNG o WebP cuadrado de 256×256 px. También puede configurarse una URL o ruta personalizada mediante `crestUrl` en `data.json`. Si una imagen no existe, el juego mostrará automáticamente las siglas del equipo.
